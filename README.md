 # 🌡️ Temperature Converter

A simple Temperature Converter built using HTML, CSS, and JavaScript. It allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin instantly.

## 🚀 Live Demo

[View Live Demo]( https://vaishnavi-payghan.github.io/Temperature-converter/)

## Features

- Convert between Celsius, Fahrenheit, and Kelvin
- Real-time conversion
- Simple and user-friendly interface
